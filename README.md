# vue-gallery
Это небольшое приложение с галереей фотографий. Нажми на кнопку Фотографии, чтобы посмотреть или добавить свои.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
